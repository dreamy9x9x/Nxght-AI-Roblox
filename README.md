# Nxght-AI-Roblox
Simple project. Definitely not a 100% real LLM, but i think you'll like it.


Essa aqui é a Nxght AI. 

cri... cri... cri... 

você: "o que isso faz?"

Eu te respondo, meu amigo! 


A Nxght AI é um projeto de um mini-assistente que consegue responder o usuário dentro do próprio Roblox, pois lá é onde ele roda, sem ocupar quase nada da sua memória ou algo do tipo! 
Ele usa uma tecnologia baseada em tecnologias reais usadas por grandes empresas pras suas IAs, mas usamos em nível totalmente simples, usando apenas versões adaptadas de coisas como a tokenização, geração, semântica, etc. 
Lembrando que por ser pequeno, ele apenas dá respostas curtas e não tem um database de informações, ou seja, se perguntar algo ele provavelmente mal vai te responder. 
O intuito do projeto é fazer com que seja possível fazer não só o que já preparamos, é pra fazer além, então POR FAVOR, faz um fork desse projeto e começa a fazer suas próprias mudanças.
